# Googleカレンダー同期 — セットアップ手順

## 予約ルール（自動）

| 項目 | 設定 |
|------|------|
| 業務日 | **月〜金**（土日は不可） |
| 祝日 | **日本の祝日カレンダー**で自動除外 |
| 業務時間 | **9:00〜19:00** |
| 1枠の長さ | **1時間** |
| 開始候補 | 9:00, 10:00, … 18:00（18:00開始は19:00終了） |
| ブロック | カレンダー上の予定と**直接重なる枠のみ**（移動バッファなし） |

---

## セットアップ

1. [script.google.com](https://script.google.com/) で新規プロジェクト
2. `Code.gs` を貼り付け
3. タイムゾーン → **東京**
4. `CALENDAR_IDS` に大学・太極拳などのカレンダーIDを追加
5. `testAvailability` を実行してログ確認（カレンダー許可）
6. **デプロイ → ウェブアプリ**（アクセス: 全員）
7. URL を `contact.html` の `data-booking-calendar-api` に設定

### 祝日カレンダー

デフォルトで `ja.japanese#holiday@group.v.calendar.google.com`（日本の祝日）を参照します。  
Googleカレンダー側で「日本の祝日」を表示にしておくと、初回許可時に読み取れます。

---

## 設定の変更

`Code.gs` の `CONFIG` を編集:

```javascript
SLOT_MINUTES: 60,        // 枠の長さ
SLOTS: ["09:00", ...],   // 開始時刻
BUSINESS_END: { h: 19, m: 0 },
```

変更後は **新しいデプロイ**（新バージョン）が必要です。
